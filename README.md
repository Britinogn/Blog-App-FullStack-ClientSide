# TechPulse Blog - Client

TechPulse is a modern tech blog platform where professionals share insights, tutorials, and discussions covering coding, cybersecurity, and emerging technologies.

## About

TechPulse provides a community-driven platform for tech enthusiasts to:
- **Read** articles on coding, cybersecurity, and emerging technologies
- **Create** and publish your own blog posts with images
- **Search** posts by title, content, author, or tags
- **Engage** with the community through comments and discussions
- **Discover** trending topics through popular tags and recent posts

The platform features a clean, modern interface with responsive design, making it easy to browse and share tech content.

## Tech Stack

### Frontend
- **Vue.js 3** - Progressive JavaScript framework
- **Vite** - Fast build tool and dev server
- **Vue Router** - Client-side routing
- **Tailwind CSS** - Utility-first CSS framework
- **Axios** - HTTP client for API requests

### Backend
- **Node.js** - JavaScript runtime for the backend server
